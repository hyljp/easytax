# easyTax 要求仕様書

## 1. 项目概要

**项目名称**: easyTax
**类型**: 桌面会计管理软件
**技术栈**: Python + CustomTkinter (GUI)
**目标用户**: 日本个人事业主 / 需要进行青色申告的用户
**当前版本**: v1.0

## 3. 全局配置 (config.json)

| 配置项 | 说明 |
|--------|------|
| `THIS_YEAR` | 当前会计年度（如 "2025"） |
| `勘定種別` | 勘定科目分类：費用、資産、負債、純資産、収益 |
| `資産` | 资产科目列表（現金、売掛金 等） |
| `負債` | 负债科目列表（買掛金、未払消費税 等） |
| `純資産` | 净资产科目列表（資本金 等） |
| `収益` | 收益科目列表（売上、受取利息 等） |
| `費用` | 费用科目列表（接待交際費、旅費交通費 等） |
| `支払方式` | 支付方式：現金、立替金、クレジットカード、口座振替、口座引き落とし |
| `仕訳帳カラム` | 仕訳帳CSV的列定义（日付、金額、借方科目、概要、貸方科目、支払方式、伝票番号、INVOICE番号） |

## 4. 功能模块详细说明

### 4.1 第一页：仕訳入力（Step 1 - 已实现）

**页面**: `invoice_page.py`
**功能**: 基于领收书图片生成仕訳帳CSV

#### 操作流程：
1. 用户选择包含领收书图片的文件夹
2. 左侧显示图片查看器，支持：
   - 前后翻页浏览图片
   - 旋转图片（270度）
   - 缩放（25% ~ 500%）
3. 右侧显示仕訳入力表单：
   - **日付** — 日期（MMdd格式，系统自动补充年份）
   - **価額** — 金额
   - **勘定種別** — 下拉选择（費用/資産/負債/純資産/収益）
   - **勘定科目** — 根据種別联动的下拉选择
   - **支払方式** — 下拉选择
   - **概要** — 备注说明
   - **INVOICE番号** — 发票号码
4. 确认后写入 `output/仕訳帳_YYYYMM.csv`
5. 原始图片移动到 `output/photo/YYYYMM/` 并重命名

#### CSV 输出格式（仕訳帳）：

| 日付 | 金額 | 借方科目 | 概要 | 貸方科目 | 支払方式 | 金額 | 伝票番号 | INVOICE番号 |
|------|------|----------|------|----------|----------|------|----------|-------------|

#### 防重复机制：
- 伝票番号重复检查 (`DuplicateVoucherError`)
- 同日付+同金額重复检查 (`DuplicateEntryError`)

---

### 4.2 第二页：帳簿生成（Step 2 - 已实现）

**页面**: `ledger_page.py`
**功能**: 从仕訳帳CSV生成総勘定元帳、月度試算表、年度試算表

#### STEP 2-1：総勘定元帳の作成
- **输入**: 包含 `仕訳帳_YYYYMM.csv` 的文件夹
- **处理**: 按勘定科目拆分，每个科目生成一个CSV
- **输出**: `{科目名}_{YYYYMM}.csv`（存放于 `output/総勘定元帳and試算表/`）
- **输出格式**: 日付、借方、貸方、伝票番号、概要

#### STEP 2-2：月度試算表の作成
- **输入**: 包含総勘定元帳CSV的文件夹
- **处理**: 汇总每个科目的借方/贷方合计
- **选项**: 可选择特定月份或"全月"
- **输出**: `試算表_YYYYMM.csv`
- **输出格式**: 科目、借方、貸方（末行为合计）

#### STEP 2-3：年度試算表の作成
- **输入**: 包含月度試算表CSV的文件夹
- **处理**: 汇总全年各月試算表数据
- **输出**: `試算表_YYYY_年度.csv`
- **输出格式**: 科目、借方、貸方（末行为合计）

#### 一括実行：
- 一个按钮 "⚡ 一括実行（全STEP）" 自动按顺序执行以上三个步骤
- 用户只需选择仕訳帳CSV文件夹，三步自动完成
- 实时显示处理状态

---

### 4.3 第三页：財務諸表（Step 3 - 已实现）

**页面**: `financial_page.py`
**功能**: 基于年度試算表生成青色申告用財務諸表

#### STEP 3-1：損益計算書の作成
- **输入**: 年度試算表CSV
- **处理**: 根据 config.json 中的収益/費用科目分类，提取对应金额
  - 収益 → 取贷方金额
  - 費用 → 取借方金额
  - 计算当期純利益 = 収益合計 - 費用合計
- **输出**: `損益計算書_YYYY.csv`（存放于 `output/財務諸表/`）
- **输出格式**:
  ```
  損益計算書, YYYY年度
  【収益の部】
  {科目}, {金额}
  収益合計, {合计}
  【費用の部】
  {科目}, {金额}
  費用合計, {合计}
  当期純利益, {净利润}
  ```

#### STEP 3-2：貸借対照表の作成
- **输入**: 年度試算表CSV
- **处理**: 根据 config.json 中的資産/負債/純資産科目分类，提取对应金额
  - 資産 → 取借方金额
  - 負債 → 取贷方金额
  - 純資産 → 取贷方金额
- **输出**: `貸借対照表_YYYY.csv`（存放于 `output/財務諸表/`）
- **输出格式**:
  ```
  貸借対照表, YYYY年度
  【資産の部】
  {科目}, {金额}
  資産合計, {合计}
  【負債の部】
  {科目}, {金额}
  負債合計, {合计}
  【純資産の部】
  {科目}, {金额}
  純資産合計, {合计}
  負債・純資産合計, {合计}
  ```

#### 一括生成：
- 一个按钮 "⚡ 一括生成（損益計算書 + 貸借対照表）" 一键生成两张表
- 支持两种试算表选择方式：
  - 直接选择CSV文件
  - 选择文件夹自动查找 `試算表_YYYY_年度.csv`

## 5. 通用功能

### 5.1 会計年度切换
- 顶部栏显示当前会計年度
- 点击可弹出下拉选择（当前年 ± 3年范围）
- 切换年度后自动重建所有页面

### 5.2 侧边栏导航
- 三个页面入口：仕訳入力、帳簿生成、財務諸表
- 高亮当前活动页面

### 5.3 输出文件夹快速打开
- 帳簿生成页和財務諸表页均提供 "📂 出力フォルダを開く" 按钮

## 6. 数据流

```
领收书图片 → [仕訳入力] → 仕訳帳CSV (按月)
                              ↓
                    [総勘定元帳生成] → 各科目元帳CSV (按月)
                              ↓
                    [月度試算表生成] → 試算表CSV (按月)
                              ↓
                    [年度試算表生成] → 試算表CSV (年度)
                              ↓
                    [財務諸表生成] → 損益計算書CSV + 貸借対照表CSV
```
