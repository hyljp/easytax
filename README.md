# easyTax

個人事業主向けの青色申告会計管理ソフトウェアです。領収書の画像から仕訳帳を作成し、総勘定元帳・試算表・財務諸表（損益計算書・貸借対照表）を自動生成します。

## 機能一覧

### Step 1: 仕訳入力
- 領収書画像を閲覧しながら仕訳データを入力
- 画像の回転・ズーム対応
- 月別の仕訳帳CSVを自動生成
- 伝票番号・同日同額の重複チェック

### Step 2: 帳簿生成
- **総勘定元帳** — 仕訳帳CSVから勘定科目ごとの元帳を生成
- **月度試算表** — 総勘定元帳から月別の試算表を生成
- **年度試算表** — 月度試算表を合算して年度試算表を生成
- 一括実行ボタンで3ステップを自動実行

### Step 3: 財務諸表（青色申告）
- **損益計算書** — 年度試算表から収益・費用を分類して自動作成
- **貸借対照表** — 年度試算表から資産・負債・純資産を分類して自動作成
- 一括生成ボタンで両方を同時に作成

## データフロー

```
領収書画像 → 仕訳帳CSV（月別）
                 ↓
           総勘定元帳CSV（科目別・月別）
                 ↓
           試算表CSV（月別）
                 ↓
           試算表CSV（年度）
                 ↓
     損益計算書CSV + 貸借対照表CSV
```

## セットアップ

### 開発環境

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

### ビルド（単一実行ファイル）

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name easyTax --icon=taxMate.ico --add-data "config.json;." main.py
```

生成された `dist/easyTax.exe` を任意の場所に配置して実行できます。`config.json` はEXE内に内蔵されており、初回起動時に自動で展開されます。

## 動作環境

- Python 3.10+
- Windows 10/11

## 設定ファイル

`config.json` で勘定科目や支払方式をカスタマイズできます。

| 項目 | 説明 |
|------|------|
| `THIS_YEAR` | 会計年度 |
| `資産` / `負債` / `純資産` / `収益` / `費用` | 各勘定科目リスト |
| `支払方式` | 支払方法の選択肢 |
| `仕訳帳カラム` | 仕訳帳CSVのカラム定義 |
